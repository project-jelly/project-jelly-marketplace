# project-jelly-marketplace

설치 가능한 **skill · plugin · MCP 설정**의 소스 저장소(마켓플레이스).
Claude Code · Codex · OpenCode에서 discovery/install 해서 쓴다.

이 리포는 *도구(tool)* 만 담는다. 배포 환경에 특정한(deployment-specific) 내용은
여기 두지 않는다 — 이 저장소는 **공개**다.

> 현재 상태: 기본 세팅만. 플러그인 미추가.

## 설치 (예정)

```
/plugin marketplace add project-jelly/project-jelly-marketplace
/plugin install <plugin>@project-jelly
```

자세한 작업 규약은 [`CLAUDE.md`](./CLAUDE.md) 참고.
